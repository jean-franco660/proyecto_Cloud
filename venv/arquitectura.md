# Documentación de Arquitectura del Proyecto

## Descripción General
Este proyecto está diseñado para practicar el uso de GitHub Actions con un flujo básico en Python.

## Componentes
- **main.py**: Contiene el script que imprime un mensaje.
- **main.yml**: Configura el flujo de CI para ejecutar pruebas y linting.

## Flujo CI/CD
1. Detecta push o pull request.
2. Configura entorno Python.
3. Instala dependencias.
4. Ejecuta pruebas y análisis de código.
